# StrayConnected 🐾

A Django-based website to help stray animals.

## Features
- 📍 Report stray dogs with photo, location & description
- 🗺️ View reported stray locations on Google Maps
- 🧾 Donate using QR code or UPI
- 📞 Contact form to reach out

## Tech Stack
- Python | Django
- HTML | CSS | JS
- Google Maps API
